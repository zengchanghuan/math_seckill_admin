### 2. 题目管理 📝

#### 功能
- ✅ 题目列表展示
- ✅ 搜索和筛选（章节、难度）
- ✅ 查看题目详情
- ✅ 新增题目
- ✅ 编辑题目
- ✅ 删除题目
- ✅ 分页功能
- ✅ **LaTeX公式实时预览**

#### 使用方式

**新增题目：**
1. 点击"新增题目"按钮
2. 填写题目内容（支持LaTeX，用`$`包围）
3. 选择类型、难度、章节
4. 如果是选择题，填写4个选项
5. 填写标准答案和解析
6. 点击"确定"

**LaTeX输入示例：**
```
题目：计算: $\sin(30^\circ) = ?$
答案：$\frac{1}{2}$
选项A：$\frac{1}{2}$
选项B：$\frac{\sqrt{2}}{2}$
```

**搜索筛选：**
- 章节下拉选择
- 难度下拉选择
- 点击"搜索"查询

## 📐 LaTeX公式使用

### 常用语法

```latex
# 分数
\frac{1}{2}
\frac{\sqrt{3}}{2}

# 根号
\sqrt{2}
\sqrt[3]{8}

# 角度
30^\circ
\pi/6

# 三角函数
\sin(x)
\cos(x)
\tan(x)

# 上下标
x^2
a_n

# 希腊字母
\alpha
\beta
\theta

# 积分
\int_0^\infty

# 求和
\sum_{i=1}^n
```

### 在管理后台中使用

**题目内容：**
```
计算: $\sin(30^\circ) = ?$
```

**选项：**
```
A: $\frac{1}{2}$
B: $\frac{\sqrt{2}}{2}$
C: $\frac{\sqrt{3}}{2}$
D: $1$
```

**答案：**
```
$\frac{1}{2}$
```

系统会自动渲染为漂亮的数学公式！
